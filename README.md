# Математическая статистика

Qt-приложение для расчёта по математической статистике (PySide6).

## Структура проекта

```
matstat/
├── main.py                 # Точка входа
├── README.md
└── src/
    ├── __init__.py
    ├── data_parser.py      # Парсинг исходных данных
    ├── calculator.py       # Статистические расчёты (класс StatisticsCalculator)
    ├── plotter.py          # Отрисовка графиков (MplCanvas + функции)
    └── gui.py              # Графический интерфейс (MainWindow)
```

## Функциональность

Приложение реализует следующие шаги:

| Шаг | Описание |
|-----|----------|
| 2 | Размах выборки: x_min, x_max, R |
| 3 | Число интервалов k = ⌊1 + 3.32·lg(n)⌋ |
| 4 | Шаг интервала h = R/k |
| 5 | Границы интервалов |
| 6 | Гистограмма и эмпирическая функция распределения |
| 7 | Статистическая таблица (частоты, накопленные частоты) |
| 8 | Среднее и исправленная дисперсия для сгруппированных данных |
| 9 | Проверка нормальности (критерий Шапиро-Уилка) |
| 10 | Проверка гипотезы о дисперсии (χ²-критерий) |
| 11 | Проверка гипотезы о мат. ожидании (t-критерий Стьюдента) |

## Зависимости

- Python ≥ 3.10
- PySide6
- matplotlib
- numpy
- scipy

## Запуск

```bash
pip install PySide6 matplotlib numpy scipy
python main.py
```
